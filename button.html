Quick Notes

  • btn is the base button class.
  • Use contextual classes like btn-primary, btn-success, etc.
  • Button sizes: btn-lg, default, btn-sm.
  • Use w-100 for a full-width button.
  • Use the disabled attribute to disable a button.
  • btn-group creates horizontal button groups.
  • btn-group-vertical creates vertical button groups.
  • Bootstrap 5 removed btn-default, btn-xs, btn-block, and btn-group-justified.
