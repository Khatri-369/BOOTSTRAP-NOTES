Bootstrap Dropdowns & Icons

Bootstrap provides dropdown components to create menus inside buttons. Bootstrap 5 also provides Bootstrap Icons for adding beautiful icons to buttons, forms, navigation bars, alerts and many other components.

1. Nesting Button Groups & Dropdown Menus

A nested button group means placing one btn-group inside another btn-group. It is mainly used to create dropdown menus inside a button group.

2. Split Button Dropdown

A Split Button Dropdown separates the main action button from the dropdown button. Clicking the main button performs the default action while clicking the arrow opens the dropdown menu.

  • HEADPHONE
  • MOBILE
  • TV

Explanation: Clicking the Sony button performs the main action. Clicking the arrow button displays additional options.

3. Bootstrap Icons

Bootstrap Icons are a collection of free SVG icons that can be used inside buttons, forms, navigation bars and many other Bootstrap components.

Syntax

<i class="bi bi-search"></i>
<i class="bi bi-download"></i>
<i class="bi bi-trash"></i>
<i class="bi bi-person-fill"></i>

Explanation: The first class bi enables Bootstrap Icons and the second class specifies the icon name.

4. Common Bootstrap Icons

Icon Class Purpose
bi-search Search
bi-house-door-fill Home

Quick Notes

  • btn-group groups multiple buttons together.
  • Nested btn-group creates a dropdown inside another button group.
  • dropdown-toggle creates a dropdown button.
  • dropdown-toggle-split creates a split dropdown button.
  • dropdown-menu stores dropdown items.
  • Bootstrap 5 uses Bootstrap Icons.
  • Add Bootstrap Icons using the <i> tag.
  • Always include the Bootstrap Icons CDN before using icons.
  • Bootstrap 3 used Glyphicons, but they were removed in Bootstrap 4 and Bootstrap 5.
